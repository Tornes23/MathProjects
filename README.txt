Included files:
	- input.m: The input data for the problem can be modified here.
	- beziercurve.m: Chooses the algorithm based on the input data.
	- directevaluation.m: Computes and plots the bezier curve using direct evaluation method.
	- decasteljau.m: Computes and plots the bezier curve using De Casteljau method.
	- midsubdivision.m: Computes and plots the bezier curve using midpoint subdivision method.
	- populate.m: Helper function to compute the midpoints recursively.

The program can be executed by writting beziercurve on the command window.
Before that, the input data can be modified in input.m
